===================================================
                 WORDMANIA
===================================================

Game "Wordmania" — guess the word from the hint!

Rules:
- You get a hint (intentionally weird or absurd).
- Type the word you think matches.
- If correct — you earn a point and continue.
- If wrong — your score resets.
- The best score is saved and shown at the start.

Controls:
- Type your answer and press Enter.
- Press Enter on the start screen to begin.
- Press Ctrl+C to exit the game.

Requirements:
- Python 3.x
- A words.txt file with "word | hint" pairs in the same folder.
- The first line in words.txt should be the best score (a number).

Have fun playing!

RU

===================================================
                ВОРДМАНИЯ (WORDMANIA)
===================================================

Игра "Вордмания" — угадай слово по подсказке!

Правила:
- Тебе показывается подсказка (но она намеренно странная или абсурдная).
- Введи слово, которое, по твоему мнению, подходит.
- Если угадал — получаешь очко и идёшь дальше.
- Если ошибся — счёт сбрасывается.
- Лучший результат сохраняется и показывается при старте.

Управление:
- Вводи слово и нажимай Enter.
- Чтобы начать игру, нажми Enter на стартовом экране.
- Для выхода из игры нажми Ctrl+C.

Требования:
- Python 3.x
- Файл words.txt с парами "слово | подсказка" в той же папке.
- В первой строке файла words.txt должен быть лучший счёт (число).

Приятной игры!
